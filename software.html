Software

While conducting my research, I have written some useful programs for the Macaulay2 computer algebra system:

  • the package 'VectorFields' provides useful functions for dealing with modules of vector fields on affine space (Lie bracket, etc.). To install it, start Macaulay2 and type: 
    		getPackage("VectorFields",Repository=>"http://www.utsc.utoronto.ca/~bpike/software/");
    to have Macaulay2 download the software and build the documentation. Before you may use it in a M2 session, you must do 
    		needsPackage "VectorFields";
  • Here are several scripts used for computing singular Milnor numbers, which were used for "Solvable Groups, Free Divisors and Nonisolated Matrix Singularities II: Vanishing Topology" (joint with James Damon).
    • calc_smn.m2 provides methods for computing singular Milnor numbers. I also provide a method implementing the Le-Greuel formula to compute the Milnor number of an ICIS.
    • mat2x3.m2 uses calc_smn.m2 to compute the vanishing Euler characteristics of 2x3 matrix singularities by the formula in the paper.
    • mat2x3verify.m2 uses mat2x3.m2 to compute the examples listed at the end of the paper.
